¿Porqué cambia el posicionamiento de las cajas internas al div principal?
Porque por defecto el orden es en filas (row), y al usar flex-direction: column, el orden cambia a columnas.

¿Qué pasa si me solicitan hacer un cambio en el orden de los elementos pero sin tocar los archivos html y js, será que puedo lograrlo a traves de
CSS?
si, usando la propiedad de la flexbox, como flex-direction u order.

¿Qué hacen estás propiedades?
flex-grow, permite que un elemento crezca para ocupar el espacio disponible en el contenedor flex.
width, define el ancho de un elemento en base al porcentaje de la pantalla.

¿Puedo diseñar toda mi web usando GRID?
Si, es posible diseñar toda una web usando grid, pero no es lo mas recomendable.

Porqué se dan estos cambios entre tan pocas líneas de diseño CSS
Porque al usar el display grid, se pueden definir areas especificas para cada elemento, y esos espacios son sencillos de definir y manipular.
